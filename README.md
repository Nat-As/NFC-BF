# NFC-Brute Forcer

## *Linux*
1. git clone https://github.com/Nat-As/NFC-BF.git
2. gcc NFC-BF.c
3. ./NFC-BF
4. Ctrl+C to exit

## *Windows*
1. Download Zip
2. Use the .exe file
3. Note: Program is currently 16 bit
